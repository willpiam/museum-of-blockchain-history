# Museum of Blockchain History 

Museum of Blockchain History is a simple site that highlights notable on-chain transactions and links directly to their explorers.

[museum.md](/museum.md)

To regenerate `museum.md` simply run

    python3 generate_museum_md.py

To view the museum on the web visit _. 

To view the museum locally in `html` run

    python3 -m http.server 8000

And then navigate to `localhost:8000`
